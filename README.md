# 📊 Proyecto SQL: Análisis de Datos con Northwind

## 🧩 Descripción del Proyecto
Este proyecto consiste en el análisis de la base de datos **Northwind**, la cual simula las operaciones de una empresa de distribución.

El objetivo es responder preguntas de negocio mediante consultas SQL, aplicando técnicas de análisis de datos para extraer información relevante sobre ventas, clientes y rendimiento comercial.

---

## 🎯 Objetivo
Desarrollar consultas SQL que permitan interpretar datos desde una perspectiva analítica utilizando:

- Funciones agregadas (SUM, AVG, COUNT)
- JOINs
- Subconsultas
- CTEs (Common Table Expressions)
- Funciones de ventana
- Optimización de consultas (EXPLAIN)

---

## 🗂️ Estructura de la Base de Datos

Las principales tablas utilizadas son:

- **Customers** → Información de clientes  
- **Employees** → Empleados que gestionan pedidos  
- **Products** → Productos y precios  
- **Categories** → Clasificación de productos  
- **Orders** → Pedidos realizados  
- **OrderDetails** → Detalles de cada pedido  
- **Shippers** → Empresas de envío  
- **Suppliers** → Proveedores  

---

## 📈 Análisis Realizados

### 🔹 Análisis de Ventas
- Top 10 productos con mayores ingresos  
- Categorías más rentables  
- Cliente con mayor gasto total  
- País con mayores ingresos  
- Empleado con más pedidos gestionados  

### 🔹 Comportamiento de Clientes
- Porcentaje de clientes sin compras  
- Países con más clientes activos  
- Clientes con mayor diversidad de compras  
- Análisis de preferencia por categoría  
- Ticket promedio por pedido  

### 🔹 Rendimiento y Tendencias
- Ranking de clientes por gasto  
- Ventas mensuales  
- Mejor mes en ventas  
- Evolución del gasto por cliente  
- Productos más vendidos por trimestre
